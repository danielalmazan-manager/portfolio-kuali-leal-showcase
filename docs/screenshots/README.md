## 📐 Screenshot Guidelines

### Technical Requirements
- **Format**: PNG (for UI) or JPG (for photos)
- **Resolution**: Minimum 1920x1080 for desktop, 375x812 for mobile
- **File size**: Compress to < 500KB per image
- **Naming**: Use descriptive names (e.g., `dashboard-analytics-overview.png`)

### Content Guidelines
- Show complete workflows, not isolated screens
- Include tooltips or annotations if helpful
- Demonstrate responsive design (desktop + mobile)
- Highlight key features and interactions

---

## 🛠️ Tools for Screenshot Creation

### Recommended Tools
- **macOS**: Built-in Screenshot (Cmd+Shift+4)
- **Windows**: Snipping Tool or Snip & Sketch
- **Browser Extensions**: Awesome Screenshot, Fireshot
- **Professional**: Cleanshot X (macOS), ShareX (Windows)

### Annotation Tools
- **Arrows & highlights**: Skitch, Annotate
- **Mockups**: Screely, Mockuphone
- **Video recording**: Loom, Screen Studio
